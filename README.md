# Estadistica-Multivariante-Aplicada a la Ecología

Estadística Multivariante Aplicada: Análisis de Comunidades Ecológicas

#  Descripción:

Repositorio educativo con ejercicios prácticos de métodos multivariantes avanzados en R (PCA, CA, DCA, PERMANOVA). Diseñado para enseñanza en postgrado de Estadística, Ecología o Ciencias Ambientales.

#  Contenido:

3 módulos teóricos (PDF): Datos ambientales, relaciones especies-ambiente, gradientes ecológicos
4 ejercicios prácticos (R):

## Ejercicio 1 -Ordenación: PCA vs CA

Introducción a métodos de ordenación lineal (PCA) vs unimodales (CA)
Comparación visual de estrategias de ordenación
Uso del dataset estándar "dune" (comunidad de plantas)

## Ejercicio 2 -Criterios metodológicos con DCA (longitud de gradiente)

Implementación de Detrended Correspondence Analysis (DCA)
Metodología clave: Uso de longitud de gradiente para decidir entre métodos lineales (PCA/RDA) o unimodales (CA/CCA)
Criterios prácticos: <3 SD (lineal), >4 SD (unimodal), 3-4 SD (evaluación ambos)

## Ejercicio 3 - Visualización: biplots y triplots

Generación de Biplots, Biplots de sitios-especies y Triplots
Integración de variables ambientales mediante envfit
Análisis visual de agrupamientos con elipses de confianza

## Ejercicio 4 - Análisis de clusters y pruebas PERMANOVA

Clustering jerárquico (método Ward) y k-means sobre ordenaciones
Pruebas de significancia PERMANOVA
Validación estadística de diferencias entre grupos

## Tecnologías y Bibliotecas:

Plataformas RStudio, Google Colab, Visual Studio Code, con Leniaje R con paquetes especializados: vegan, ggplot2, ggrepel, cluster, indicspecies
Para la construccipon de Proyectos de Investigacion aplicando Estadistica con R.
Enfoque reproducible y documentado para enseñanza.

## Competencias Desarrolladas en el Estudiante:

✓ Diseño and ejecución de Análisis Multivariantes
✓ Interpretación de Ordenaciones Ecológicas
✓ Visualización Avanzada de Datos Complejos
✓ Toma de Decisiones Metodológicas Basadas en Características de Datos
✓ Pruebas de Hipótesis no Paramétricas
✓ Programación Reproducible en Ecología Estadística
